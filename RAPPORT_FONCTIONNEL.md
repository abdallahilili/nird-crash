# Rapport Fonctionnel - NIRD Crash

## Vue d'ensemble

NIRD Crash est une application éducative et ludique qui permet aux utilisateurs de découvrir le Nouvel Institut de Recherche Digitale (NIRD) tout en s'amusant. L'application combine apprentissage et jeu pour rendre la découverte du numérique responsable accessible et engageante.

---

## 1. Page d'accueil

### 1.1 Présentation générale
Lorsque l'utilisateur ouvre l'application, il arrive sur une page d'accueil accueillante qui présente le NIRD.

**Fonctionnalités disponibles :**
- **Titre et description** : L'application affiche clairement son nom "NIRD Crash" et explique qu'il s'agit d'un jeu interactif pour découvrir le Nouvel Institut de Recherche Digitale
- **Bouton de démarrage** : Un bouton visible permet de commencer l'aventure et d'accéder aux niveaux de jeu

### 1.2 Vidéo de présentation
Une section vidéo est disponible sur la page d'accueil.

**Fonctionnalités :**
- **Lecture de vidéo** : L'utilisateur peut regarder une vidéo qui présente la mission et les valeurs du NIRD
- **Contrôles vidéo** : Des boutons permettent de mettre en pause ou de relancer la vidéo
- **Description** : Un texte explique que la vidéo présente la mission et les valeurs de l'organisation

### 1.3 Informations sur le NIRD
Une section dédiée explique ce qu'est le NIRD et ses valeurs.

**Contenu disponible :**
- **Description de l'organisation** : Explication de ce qu'est le Nouvel Institut de Recherche Digitale
- **Mission** : Présentation de la mission de l'organisation
- **Valeurs** : Liste des valeurs importantes :
  - Innovation
  - Collaboration
  - Durabilité
  - Excellence
  - Impact

### 1.4 Avantages de jouer
Une section présente les raisons de jouer à NIRD Crash.

**Points mis en avant :**
- Découvrir le NIRD et ses valeurs
- Apprendre en jouant
- Rejoindre une communauté
- Explorer l'innovation digitale

## 2. Carte de progression

### 2.1 Vue d'ensemble
Après avoir cliqué sur "Commencer l'aventure", l'utilisateur accède à une carte interactive qui représente son parcours d'apprentissage.

**Fonctionnalités principales :**
- **Deux modes d'affichage** : L'utilisateur peut choisir entre deux onglets :
  - **Éducation** : Pour consulter des contenus éducatifs
  - **Quiz & Jeux** : Pour accéder aux niveaux de jeu

### 2.2 Mode Éducation
Dans ce mode, la carte affiche des points d'intérêt éducatifs.

**Fonctionnalités :**
- **Points cliquables** : Plusieurs points sont visibles sur la carte, chacun représentant une section éducative
- **Informations au survol** : Quand l'utilisateur passe sa souris sur un point, il voit le titre et le sous-titre de la section
- **Détails complets** : En cliquant sur un point, une fenêtre s'ouvre avec :
  - Le titre de la section
  - Le contenu éducatif détaillé
  - Des liens vers des ressources supplémentaires si disponibles
  - Un bouton pour aller directement aux quiz correspondants

**Sections éducatives disponibles :**
1. Pourquoi agir ? - L'urgence du numérique responsable
2. Qu'est-ce que NIRD ? - Un collectif pour le numérique scolaire
3. Les 3 piliers de NIRD - Inclusion, Responsabilité, Durabilité
4. Concrètement, que fait NIRD ? - Actions concrètes sur le terrain
5. Qui peut agir ? - Tous les acteurs du numérique éducatif
6. Comment rejoindre - Informations pour participer
7. Exemple inspirant - Cas pratiques
8. Logiciels libres - Ressources disponibles

### 2.3 Mode Quiz & Jeux
Dans ce mode, la carte affiche tous les niveaux de jeu disponibles.

**Fonctionnalités :**
- **Niveaux accessibles** : Tous les niveaux sont disponibles dès le départ
- **Informations visuelles** : Chaque niveau affiche :
  - Son numéro
  - Son thème (couleur distinctive)
  - Son état de complétion (terminé ou non)
  - Le nombre d'étoiles obtenues si le niveau est complété
- **Informations au survol** : En passant la souris sur un niveau, l'utilisateur voit :
  - Le titre du niveau
  - Le thème
  - Le score obtenu et le nombre de mots trouvés (si complété)
- **Sélection** : Un clic sur un niveau lance le jeu

**Niveaux disponibles :**
L'application propose 12 niveaux thématiques différents, chacun abordant un sujet lié au numérique éducatif :
- Définition NIRD
- Microsoft Éducation
- Google Classroom
- Apple School
- Facebook Campus
- Amazon Edu
- Tesla Tech School
- Netflix & Learn
- TikTok University
- Uber for Tutors
- Zoom School
- Blockchain Academy

## 3. Jeu principal

### 3.1 Interface de jeu
Une fois un niveau sélectionné, l'utilisateur accède à l'écran de jeu.

**Éléments visibles :**
- **Bouton retour** : Permet de revenir à la carte de progression
- **Titre du niveau** : Affiche clairement le niveau en cours
- **Barre de progression** : Montre le score actuel, l'objectif à atteindre et les étoiles gagnées
- **Personnage animateur** : Un personnage réagit aux actions du joueur
- **Grille de lettres** : La zone principale où se déroule le jeu

### 3.2 Grille de lettres
Le cœur du jeu consiste à trouver des mots dans une grille de lettres.

**Fonctionnement :**
- **Grille interactive** : Une grille de lettres est affichée (généralement 5x5)
- **Sélection de lettres** : L'utilisateur peut cliquer sur les lettres pour former un mot
- **Validation** : Une fois un mot formé, l'utilisateur peut le valider
- **Feedback visuel** : Les lettres sélectionnées sont mises en évidence

**Résultats possibles :**
- **Mot correct** : Si le mot fait partie de la liste des mots à trouver :
  - Le mot est ajouté à la liste des mots trouvés
  - Des points sont attribués
  - Des effets visuels apparaissent (particules, animations)
  - Un message de félicitation s'affiche
  - Le personnage réagit positivement
- **Mot incorrect** : Si le mot n'est pas dans la liste :
  - Un message d'erreur s'affiche
  - Le personnage réagit négativement
  - Un message d'encouragement apparaît

### 3.3 Informations sur les mots
Quand un mot est trouvé, une fenêtre s'ouvre avec des informations.

**Contenu de la fenêtre :**
- **Le mot trouvé** : Affiché en grand
- **Explication** : Une description amusante et éducative du mot
- **Points gagnés** : Le nombre de points attribués pour ce mot
- **Image** : Une illustration associée au mot (si disponible)

### 3.4 Système de points et combos
L'application récompense la rapidité et la régularité.

**Système de points :**
- Chaque mot trouvé rapporte des points
- Les points varient selon la difficulté du mot
- Le score total est affiché en permanence

**Système de combos :**
- Si l'utilisateur trouve plusieurs mots rapidement (dans les 5 secondes), un combo s'active
- Les combos multiplient les points gagnés
- Un message spécial apparaît pour indiquer le combo
- Des effets visuels renforcent l'effet du combo

### 3.5 Système d'étoiles
La performance est récompensée par des étoiles.

**Attribution des étoiles :**
- Le nombre d'étoiles dépend du nombre de mots trouvés
- 1 étoile : Objectif minimum atteint
- 2 étoiles : Bonne performance
- 3 étoiles : Performance excellente (tous les mots trouvés)

**Affichage :**
- Les étoiles sont visibles dans la barre de progression en haut
- Elles s'animent quand elles sont gagnées

### 3.6 Questions de quiz
Pendant le jeu, des questions apparaissent de manière aléatoire.

**Types de questions :**

**Questions à compléter :**
- Une phrase avec un mot manquant
- Plusieurs choix de réponse proposés
- L'utilisateur sélectionne la bonne réponse
- Points bonus attribués en cas de bonne réponse

**Questions avec lettres intermittentes :**
- Un mot est affiché avec certaines lettres manquantes
- L'utilisateur doit deviner le mot complet
- Points bonus plus élevés pour ce type de question

**Fonctionnement :**
- Les questions apparaissent entre les recherches de mots
- Après avoir répondu, un bouton permet de passer à la question suivante
- Le personnage réagit selon que la réponse est correcte ou non

### 3.7 Messages d'encouragement
L'application encourage l'utilisateur tout au long du jeu.

**Types de messages :**
- Messages de félicitation quand un mot est trouvé
- Messages motivants aléatoires
- Messages spéciaux lors des combos
- Messages d'aide si l'utilisateur a des difficultés

### 3.8 Fin de niveau
Quand l'utilisateur a trouvé suffisamment de mots, le niveau se termine.

**Écran de fin de niveau :**
- **Confettis** : Des animations de célébration apparaissent
- **Fenêtre de résultats** : Une fenêtre s'affiche avec :
  - Le score final
  - Le nombre d'étoiles obtenues
  - Le nombre de mots trouvés
  - Le nombre total de mots disponibles
- **Options** : L'utilisateur peut :
  - Passer au niveau suivant
  - Rejouer le niveau
  - Retourner au menu principal

## 4. Système de badges

### 4.1 Principe
L'application récompense les accomplissements avec des badges.

**Fonctionnement :**
- Les badges se débloquent automatiquement quand les conditions sont remplies
- Une notification apparaît quand un badge est obtenu
- Les badges sont visibles dans le profil de l'utilisateur

### 4.2 Badges disponibles

**Innovateur**
- Condition : Compléter 3 niveaux sur l'innovation et les idées
- Description : Récompense l'exploration des concepts innovants

**Créateur**
- Condition : Trouver 50 mots au total
- Description : Récompense l'assiduité dans la recherche de mots

**Contributeur**
- Condition : Compléter 5 niveaux sur la collaboration et la communauté
- Description : Récompense l'engagement communautaire

**Résolveur**
- Condition : Résoudre toutes les énigmes (12 niveaux)
- Description : Récompense la persévérance

**Expert NIRD**
- Condition : Obtenir 3 étoiles sur tous les niveaux
- Description : Récompense l'excellence

**Écologiste**
- Condition : Compléter tous les niveaux environnement
- Description : Récompense l'intérêt pour l'environnement

**Esprit Communautaire**
- Condition : Compléter 20 niveaux au total
- Description : Récompense la participation active

**Maître des Mots**
- Condition : Trouver 100 mots au total
- Description : Récompense la maîtrise du jeu

## 5. Suivi de progression

### 5.1 Score global
L'application garde en mémoire le score total de l'utilisateur.

**Fonctionnalités :**
- Le score s'affiche dans l'en-tête de l'application
- Il s'accumule au fil des parties
- Il permet de débloquer certains badges

### 5.2 Historique des niveaux
L'application enregistre la progression sur chaque niveau.

**Informations sauvegardées :**
- Niveaux complétés
- Score obtenu sur chaque niveau
- Nombre d'étoiles gagnées
- Mots trouvés

**Affichage :**
- Sur la carte, les niveaux complétés sont visuellement distincts
- Les statistiques sont visibles au survol des niveaux
